# 📰 Fake News Detector

A Machine Learning web application that classifies news articles as **Fake** or **Real** using **TF-IDF Vectorization** and **Logistic Regression**. The application is built with **Flask** and deployed online using **Render**.

## 🌐 Live Demo

**Live Website:**  
https://fake-news-detector-r6xv.onrender.com

## 📂 GitHub Repository

https://github.com/utkarshsinghcollegee-del/Fake-News-Detector

---

## 📌 Features

- 🔍 Detects Fake and Real News
- 🤖 Machine Learning Model (Logistic Regression)
- 📰 TF-IDF Text Vectorization
- ⚡ Real-time Prediction
- 📊 Displays Prediction Confidence
- 🌐 Flask Web Application
- ☁️ Live Deployment on Render
- 💻 Responsive User Interface

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/utkarshsinghcollegee-del/Fake-News-Detector.git
```

### Move into the project

```bash
cd Fake-News-Detector
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```

## 📊 Machine Learning Model

- Algorithm: Logistic Regression
- Feature Extraction: TF-IDF Vectorizer
- Dataset Size: 44,000+ News Articles
- Accuracy: ~98%
